## Coverage

These assignments comprehensively cover the following topics:

- **Computer Organization**: Register organization, instruction set architecture, addressing modes
- **Control Unit Design**: Hardwired control, micro-programmed control, control memory
- **Arithmetic Operations**: Binary arithmetic, Booth's multiplication, restoring division
- **Memory Organization**: Memory hierarchy, cache mapping (direct, associative, set-associative)
- **I/O Organization**: I/O interfaces, interrupt handling, DMA
